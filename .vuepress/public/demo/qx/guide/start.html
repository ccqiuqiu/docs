# 技术准备

开始之前，请确认您已熟练掌握以下技术或组件库

  • ES6 (opens new window)
  • vue.js (opens new window)、vuex (opens new window)、vue-router (opens new window)
  • ElementUI (opens new window)
  • form-create (opens new window)

# 环境要求

  • node.js >= 8

# 安装

千寻提供了脚手架工具 qianxun-cli 来初始化项目，您需要全局安装。

$ npm install qianxun-cli -g
1

# 初始化项目

执行以下命名，按照提示即可完成项目的初始化

$ qianxun init
1

# 目录结构

初始化后的项目结构如下

├─ api                          // api目录
│   └─ index.js                 // api综合导出文件
├─ public                       // 公共资源目录
│   ├─ favicon.ico              // 网站favicon
│   └─ index.html               // 主页
├─ src                          // 源码目录
│   ├─ assets                   // 静态资源目录
│   │   ├─ css                  // 样式文件目录
│   │   │   ├─ base.scss        // 基础样式
│   │   │   ├─ theme.scss       // 主题相关样式
│   │   │   └─ vars.scss        // 样式变量定义
│   │   ├─ icons                // 图标目录
│   │   │   ├─ svg              // svg文件目录
│   │   │   │   └─ *.svg        // svg文件
│   │   │   └─ icon.js          // 读取.svg文件
│   │   └─ utils                // 工具类/方法目录
│   │       ├─ constant.js      // 常量
│   │       ├─ CrudUtils.js     // CRUD相关的工具方法
│   │       ├─ index.js         // 统一导出工具类
│   │       ├─ lsUtils.js       // 本地存储相关的工具方法
│   │       ├─ uiUtils.js       // UI相关的工具方法
│   │       └─ utils.js         // 其他一些工具方法
│   ├─ conmponents              // 业务无关的基础组件
│   │   ├─ *.vue                // 组件文件
│   ├─ router                   // 路由配置
│   │   ├─ demo.js              // 路由模块
│   │   └─ index.js             // 路由的集合导出文件
│   ├─ store                    // vuex配置
│   │   ├─ system.js            // vuex命名模块
│   │   └─ index.js             // vuex集合导出文件
│   └─ views                    // 视图组件目录
│   │   ├─ demo                 // demo模块
│   │   │   ├─ fragment         // vue片段目录
│   │   │   │   └─ *.vue        // .vue片段文件，主要用于细化组件
│   │   │   └─ *.vue            // page类型的组件
│   │   └─ ......               // 其他一些模块，每个模块的目录和文件同common一致
│   └─ main.js                  // 入口文件
├─ .editorconfig                // 编辑器配置
├─ .eslintrc.js                 // eslint配置
├─ .env.development             // development环境的环境变量配置
├─ .env.production              // 生产环境的环境变量配置
├─ .gitignore                   // git忽略文件
├─ babel.config.js              // babel配置文件
├─ package.json                 // 项目的npm配置文件
├─ package-lock.json            // npm版本锁
├─ README.md                    // readme
└─ veu.config.js                // vue-cli配置
